# Global AP Control & Compliance Framework (Excel)

## Overview
An end-to-end Accounts Payable control model for multi-country operations, combining validation rules, payment batch governance, EOR reconciliation, and donor/grant compliance monitoring.

## Flow
Raw_Input → Validation_Engine → Payment_Batch → EOR_Reconciliation → Grant_Register → Donor_Compliance_Summary → AP_Control_Dashboard

## Key Modules
- Validation Engine (blocking + error reasons)
- Payment Batch (approved-only execution)
- EOR Reconciliation (invoice vs payroll expectation)
- Grant Register (payments + reporting status)
- Donor Compliance Summary (% reported + compliance status)
- Executive Dashboard (controls + risk visibility)
