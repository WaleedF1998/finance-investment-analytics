# Performance Management Dashboards

## Business Question
Are teams and departments performing in line with plan and forecast?

## Decision Use
Used in monthly and quarterly business reviews to:
- track plan vs actual performance
- monitor forecast updates
- identify areas requiring corrective action

## Key Metrics
- Actual vs budget
- Forecast vs actual
- Variance by department or cost center
- Key operational KPIs

## Dashboard Design
- Executive summary view
- Variance highlights
- Drill-down for analysis when needed

## Tools
Tableau, Power BI
