# AP & Expense Control Framework

## Business Question
How can vendor spend and operating expenses be controlled without slowing down the business?

## Decision Use
Used by finance teams to:
- monitor vendor balances
- enforce approval and documentation rules
- ensure expenses remain within budget

## Scope
- Accounts payable tracking
- Expense categorization
- Budget vs actual monitoring

## Controls & Validation
- Approval workflow checks
- Missing receipt flags
- Variance alerts against budget

## Outputs
- Vendor balance summaries
- Expense variance reports
- Exception lists for follow-up

## Tools
Excel

## Status
Framework documented with partial implementation.
