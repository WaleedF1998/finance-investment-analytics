# UAE Portfolio & Risk Analysis

## Investment Question
How does a UAE real estate portfolio perform at an aggregate level, and where are the key risks?

## Decision Use
Supports portfolio-level decisions by:
- evaluating overall return and income stability
- identifying risk concentration across assets
- stress-testing downside scenarios

## Methodology
- Portfolio aggregation of NOI and cash flows
- Exposure analysis by asset type and location
- Scenario and sensitivity analysis

## UAE-Specific Considerations
- Geographic concentration (e.g., Dubai vs Abu Dhabi)
- Asset type mix (residential, mixed-use)
- Market cycle exposure

## Risk Assessment
- Concentration risk
- Market downturn sensitivity
- Cash flow volatility

## Outputs
- Portfolio-level return metrics
- Risk concentration summaries
- Stress-test results

## Tools
Excel, Tableau / Power BI

## Status
Framework documented. Portfolio model under development.
