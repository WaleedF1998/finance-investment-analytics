# Financial Anomaly Detection

## Business Question
Can unusual financial patterns be detected automatically to support early investigation?

## Decision Use
Helps finance teams:
- detect unexpected movements
- prioritize investigation efforts
- reduce reliance on manual checks

## Analytical Approach
- Statistical baselines
- ML-based anomaly detection techniques
- Comparison against historical behavior

## Validation & Controls
- Threshold calibration
- Human review of flagged anomalies
- Prevention of false positives

## Outputs
- Anomaly alerts
- Exception lists for review
- Inputs for integrity dashboards

## Tools
Python
